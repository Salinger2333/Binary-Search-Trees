# Binary Search Trees (BST)

## English

### Introduction
This project provides a comprehensive implementation of a balanced Binary Search Tree (BST) in JavaScript. It is designed to demonstrate core data structure concepts and algorithms associated with BSTs.

### Features
- **Tree Construction**: Build a balanced tree from an array of numbers (handles duplicates and sorting).
- **Basic Operations**: `insert`, `delete`, and `find` nodes.
- **Traversals**: Level-order (BFS), Pre-order, In-order, and Post-order (DFS) traversals.
- **Tree Analysis**: Calculate `height` and `depth` of nodes, and check if the tree `isBalanced`.
- **Rebalancing**: Functionality to `rebalance` an unbalanced tree.

## 中文 (Chinese)

### 简介
本项目提供了一个完整的 JavaScript 平衡二叉搜索树 (BST) 实现。旨在演示与 BST 相关的核心数据结构概念和算法。

### 功能
- **构建树**：从数字数组构建平衡树（自动处理去重和排序）。
- **基本操作**：`insert` (插入), `delete` (删除), 和 `find` (查找) 节点。
- **遍历**：层序遍历 (BFS), 前序、中序和后序遍历 (DFS)。
- **树分析**：计算节点的 `height` (高度) 和 `depth` (深度)，以及检查树是否平衡 (`isBalanced`)。
- **重平衡**：提供 `rebalance` 功能以重新平衡非平衡树。
